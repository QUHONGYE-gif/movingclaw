# 树莓派目录结构讲解

```text
/ (根目录 Root)
├── boot/ (或 /boot/firmware/)     <-- ⚠️ [树莓派核心] 启动配置与内核 (FAT32分区)
│   ├── config.txt                 # 核心硬件参数配置 (超频、分辨率、开启I2C/SPI)
│   ├── cmdline.txt                # Linux内核启动参数
│   ├── kernel.img                 # 操作系统内核文件
│   └── overlays/                  # 设备树覆盖文件 (.dtbo)
│
├── etc/                           <-- 🛠️ [配置中心] 系统与软件的全局配置
│   ├── apt/                       # apt包管理器配置 (修改国内镜像源 sources.list)
│   ├── network/                   # 网络接口配置
│   ├── dhcpcd.conf                # 静态IP等网络配置 (部分版本)
│   ├── fstab                      # 磁盘与分区开机自动挂载配置
│   └── ssh/                       # SSH远程登录配置
│
├── home/                          <-- 🏠 [用户数据] 普通用户的“家”
│   └── pi/                        # 树莓派默认用户目录 (新系统为你自定义的用户名)
│       ├── Documents/             # 文档
│       ├── Downloads/             # 下载
│       └── ...                    # 你的个人代码、脚本通常放在这里
│
├── var/                           <-- 📊 [动态数据] 经常变化的数据和日志
│   ├── log/                       # 系统日志 (排错必看: syslog, auth.log, dmesg)
│   ├── www/html/                  # Web服务器 (如Apache/Nginx) 的默认根目录
│   └── run/                       # 运行时数据 (同根目录下的 /run)
│
├── dev/                           <-- 🔌 [设备文件] 硬件设备的抽象
│   ├── ttyAMA0 / ttyS0            # 串口设备
│   ├── i2c-1                      # I2C总线设备
│   ├── video0                     # 摄像头设备
│   └── mmcblk0                    # SD卡/TF卡存储介质
│
├── sys/                           <-- 🧠 [虚拟文件系统] 内核与硬件底层视图
│   └── class/
│       ├── gpio/                  # 可通过直接读写文件来控制的GPIO引脚
│       └── leds/                  # 控制树莓派板载LED状态灯
│
├── proc/                          <-- ⚙️ [虚拟文件系统] 运行进程与系统实时状态
│   ├── cpuinfo                    # CPU硬件信息 (可查看架构、型号)
│   ├── meminfo                    # 系统内存实时使用状态
│   └── device-tree/               # 当前系统加载的设备树信息
│
├── usr/                           <-- 📦 [用户资源] 最庞大的应用程序与库目录
│   ├── bin/                       # 用户安装的可执行命令 (如 python3, git)
│   ├── local/                     # 用户自行编译安装的软件目录
│   └── lib/                       # 应用程序调用的库文件
│
├── bin/                           # 基础系统命令 (ls, cp, mkdir 等)
├── sbin/                          # 系统管理员专用命令 (ifconfig, fdisk, reboot 等)
├── lib/                           # 系统级共享库与内核模块 (底层驱动)
├── media/                         # U盘、移动硬盘插入后的自动挂载点
├── mnt/                           # 系统管理员的手动挂载点 (如挂载NAS、网络共享盘)
├── opt/                           # 第三方大型或非标准软件安装目录
├── root/                          # 超级管理员 (root) 的专属主目录
└── tmp/                           # 临时文件目录 (系统重启时通常会被清空)
```
